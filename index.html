Sparta Core Project 1 - Instructions

How to play

The keys will flash purple then return to light blue

You need to click the keys in the order they are presented

The sequence will start from the beginning each time.

Tips

Be patient, don't click too soon after you finish a round

The keys go up in pitch from left to right, learn to associate each key with the corresponsing sound

Keyboard / chillout mode

Use the 'a' 's' 'd' 'f' and 'g' keys to play the sounds without starting the game
